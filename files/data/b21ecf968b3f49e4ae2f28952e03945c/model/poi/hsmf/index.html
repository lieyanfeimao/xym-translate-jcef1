Overview(${t_2c84e5d18f6349b39cfa060fa8421ccb})

HSMF is the POI Project's pure Java implementation of the Outlook MSG format.(${t_85678915212a4dbbabdfc771c793e9a4})

At this time, it provides low-level read access to all of the file, along with a user-facing way to get at the common textual content of MSG files. to all(${t_4fff6db4b20c41a380b2290d8b89a782})

There is an example MSG textual renderer, which shows how to access the common parts such as sender, subject, message body and examples. This is in the HSMF examples area of SVN. You may also wish to look at the unit tests for more use guides.(${t_4eacaffcd9e44289867e7d7aa9569d89})
