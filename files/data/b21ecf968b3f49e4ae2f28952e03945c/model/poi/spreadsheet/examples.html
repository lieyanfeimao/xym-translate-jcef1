HSSF and XSSF common examples(${t_f6975ec4aa364500ac0357b6b45a04ee})

Apache POI comes with a number of examples that demonstrate how you can use the POI API to create documents from "real life". The examples below based on common XSSF-HSSF interfaces so that you can generate either *.xls or *.xlsx output just by setting a command-line argument:(${t_c0e33b0ffc6943568f3314bdbe5508f9})

BusinessPlan -xls
or
BusinessPlan -xlsx

All sample source is available in SVN(${t_bf7d674c709e43e1ab2f71fdc8b4534f})

In addition, there are a handful of HSSF only and XSSF only examples as well.(${t_a10c61f59e2748338b7db7840e20678c})

Business Plan(${t_7325cf904eab4b1d8705156936a9ea38})

The BusinessPlan application creates a sample business plan with three phases, weekly iterations and time highlighting. Demonstrates advanced cell formatting (number and date formats, alignments, fills, borders) and various settings for organizing data in a sheet (freezed panes, grouped rows).(${t_86686f9707494c8ea6cbed65a094f12f})

Calendar(${t_84155b4cbaa441459321be390584a65c})

The Calendar demo creates a multi sheet calendar. Each month is on a separate sheet.(${t_865a7f8d52be41d7ae15fbfff4c9ec1f})

Loan Calculator(${t_6b38b14407b64f73ae6a21355e376a01})

The LoanCalculator demo creates a simple loan calculator. Demonstrates advance usage of cell formulas and named ranges.(${t_8cbef729f5204862b29a478546285d39})

Timesheet(${t_554c94b9d12d473bb5db4ffea5c13d34})

The Timesheet demo creates a weekly timesheet with automatic calculation of total hours. Demonstrates advance usage of cell formulas.(${t_50056cb616d348f0a8b54c72efbb87dd})

Conditional Formats(${t_367df5eb87d1462b9753a017738e7d27})

  • Hide the duplicate values(${t_1bf921eaa87b42399291e33bb45f46c9})
  • Highlight duplicate entries in a column(${t_de8e94187d914f1a9b3bf6e9aa7691ab})
  • Highlight items that are in a list on the worksheet(${t_3bf9cf38755b4d428c13767948d96375})
  • Highlight payments that are due in the next thirty days(${t_6eee8b23bf5446cf81b386f9cc4dc691})
  • Shade alternating rows on the worksheet(${t_e466a49aad8d4d25b9c5662e5c71022d})

Formula Examples(${t_e64688a0bd2b447b985388a02da93005})

The CalculateMortgage example demonstrates a simple user-defined function to calculate principal and interest.(${t_803639e7604041a48c2b5e19da503849})

The CheckFunctionsSupported example shows how to test what functions and formulas aren't supported from a given file.(${t_faa90935bdb8409dac1e8ad017c924b1})

The SettingExternalFunction example demonstrates how to use externally provided (third-party) formula add-ins.(${t_099e87002a214ba3a2eaa33ff9c98c6a})

The UserDefinedFunctionExample example demonstrates how to invoke a User Defined Function for a given Workbook instance using POI's UDFFinder implementation.(${t_e74dc8df77a544f995dac2ebf0ad4f24})

Add Dimensioned Image(${t_7ed6b975919d4df382b106ddf6bc7d2f})

The AddDimensionedImage example demonstrates how to add an image to a worksheet and set that images size to a specific number of millimetres irrespective of the width of the columns or height of the rows.(${t_075bccdd8b9340b4a44ea8a2f23fc478})

Aligned Cells(${t_6cac0cbb87ce4b8c80e9874b76da90f0})

The AligningCells example demonstrates how various alignment options work.(${t_8c9258b0ac5a43d4a7b2eed4f7c96787})

Cell Style Details(${t_33767e862ded4184b5fdee3c3c265887})

The CellStyleDetails example demonstrates how to read excel styles for cells.(${t_717f12c90ed447c38a0cb5aa16a327c3})

Linked Dropdown Lists(${t_5127d78b1ab04e79a6f46a1844acd636})

The LinkedDropDownLists example demonstrates one technique that may be used to create linked or dependent drop down lists.(${t_209a661b29b24ac0ae6f13488eb64346})

Common SS Performance Test(${t_5311f0a57e3443a79a073fe34d4f0327})

The SSPerformanceTest example provides a way to create simple example files of varying sizes, and to calculate how long they take. Useful for benchmarking your system, and to also test if slow performance is due to Apache POI itself or to your own code.(${t_7108d5aa95b74256925a3db50b1f0e2b})

ToHtml(${t_eeda3ff5b2294829b8ede314bcc6cf01})

The ToHtml example shows how to display a spreadsheet in HTML using the classes for spreadsheet display.(${t_54757f6a1e21409a9bf84fe84dd19081})

ToCSV(${t_f95f6b1390a54a54a23cc62cbb44f1d9})

The ToCSV example demonstrates one way to convert an Excel spreadsheet into a CSV file.(${t_7e353c4777634b019b605e651a0fc872})
