The VT_CF Format(${t_a3122d58dc8f465784e63eec1b31d589})

Thumbnail information is stored as a VT_CF, or Thumbnail Variant. The Thumbnail Variant is used to store various types of information in a clipboard. The VT_CF can store information in formats for the Macintosh or Windows clipboard.(${t_34c0f31b8843493f948c022d38f47244})

There are many types of data that can be copied to the clipboard, but the only types of information needed for thumbnail manipulation are the image formats.(${t_7520e270d9144e379164d4af8449fa21})

The VT_CF structure looks like this:(${t_acd24a68ad274b9db2b51a3d19e9968a})

Element:(${t_300520e2bc124fbe830ee312be35d8df}) Clipboard Size(${t_7f9dba5e894b44d2b2de73e60a069768}) Clipboard Format Tag(${t_b4b902312c704539a72230c3fc2f3f19}) Clipboard Data(${t_67cdb2bea5ed44639ab17d93f2bd717e})
Size:(${t_3430faa1fcfd43018003a2e2b4687105}) 32 bit unsigned integer (DWord)(${t_aa1b0e02a2f640b2b136e5ce6b746b5a}) 32 bit signed integer (DWord)(${t_613a9d03d25a47329b0b558936ef2f94}) variable length (byte array)(${t_a2e28a8152cf4f68b117c9a03615f853})

The Clipboard Size refers to the size (in bytes) of Clipboard Data (variable size) plus the Clipboard Format (four bytes).(${t_b66557f5b2264127850821df8917f2d2})

Clipboard Format Tag has four possible values:(${t_1e4b3440f36846f1a20a823c939ef008})

Value(${t_2effaf3121254dcaa47c896ce6437ced}) Identifier(${t_71619c9262af4c308d20f2e1d55b5164}) Description(${t_338a974ea5cc48069f0294bc7e90149a})
-1L(${t_82b8dd33b81e497685f9ee7a4f06f2ad}) CFTAG_WINDOWS(${t_156d76e0a5954a04a39ebc0807f3edcd}) a built-in Windows© clipboard format value(${t_4baf854c3ad647e4acf71355d5c6d717})
-2L(${t_b135c6df78394000a13492b7c1b8fe5a}) CFTAG_MACINTOSH(${t_f594beff8c9e49c8aee6e74ecb2c5626}) a Macintosh clipboard format value(${t_48626f6392b64cf8aa2e894e157fa215})
-3L(${t_f235f1f2e8994e22bdc32cfca1f783e3}) CFTAG_FMTID(${t_25ecf69d2291440296b2833ae71a9a5a}) a format identifier (FMTID) This is rarely used.(${t_4465e02cce824162a58a976d62862fb7})
0L(${t_8356a6f77d3d495cbf73bd856beef450}) CFTAG_NODATA(${t_bf3750ec11ee421a9865ec51fe909dc7}) No data This is rarely used.(${t_536b755d36924be38f66a680af15f1a0})

Windows Metafile Format(${t_ae507971ac0c435fad19b1dc1a07dffd})

The most common format for thumbnails on the Windows platform is the Windows metafile format. The Clipboard places and extra header in front of a the standard Windows Metafile Format data.(${t_df67fd9df7794abbb3c44237c31ffd0a})

The Clipboard Data byte array looks like this when an image is stored in Windows' Clipboard WMF format.(${t_3b986ab5dbd94a41ab0fa907b3517f00})
