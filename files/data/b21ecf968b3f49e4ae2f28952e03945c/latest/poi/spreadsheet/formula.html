Formula Support(公式支持)

Introduction(介绍)

This document describes the current state of formula support in POI. The information in this document currently applies to the 3.13 version of POI. Since this area is a work in progress, this document will be updated with new features as and when they are added.(本文档描述了 POI 中公式支持的当前状态。本文档中的信息目前适用于 POI 3.13 版本。由于该领域正在进行中，因此本文档将在添加新功能时进行更新。)

The basics(基础)

In org.apache.poi.ss.usermodel.Cell setCellFormula("formulaString") is used to add a formula to a sheet, and getCellFormula() is used to retrieve the string representation of a formula.(在 org.apache.poi.ss.usermodel.Cell 中，setCellFormula("formulaString") 用于向工作表添加公式，而 getCellFormula() 用于检索公式的字符串形式。)

We aim to support the complete excel grammar for formulas. Thus, the string that you pass in to the setCellFormula call should be what you expect to type into excel. Also, note that you should NOT add a "=" to the front of the string.(我们的目标是支持公式的完整 excel 语法。因此，您传递给 setCellFormula 调用的字符串应该是您希望在 Excel 中键入的内容。另外，请注意，您不需要在字符串的前面添加“=”。)

Please note that localized versions of Excel allow to enter localized function-names. However internally Excel stores the English names and thus POI only supports these and not the localized ones. Also note that only commas may be used to separate arguments, as per the Excel English style, alternate delimeters used in other localizations are not supported.(请注意，本地化版本的 Excel 允许输入本地化的函数名称。然而，Excel内部存储英文名称，因此POI只支持英文名称，而不支持本地化名称。另请注意，根据Excel英语风格，只有逗号可以用于分隔参数，不支持在本地化中使用的其他替代参数。)

Supported Features(支持的功能)

  • References: single cell & area, 2D & 3D, relative & absolute(参考：单个单元格和面积、2D 和 3D、相对和绝对)
  • Literals: number, text, boolean, error and array(字面常量：数字、文本、布尔值、错误和数组)
  • Operators: arithmetic and logical, some region operators(运算符：算术和逻辑，一些区域运算符)
  • Built-in functions: over 350 recognised, 280 evaluatable(内置函数：350多个已识别，280个可评估)
  • Add-in functions: 24 from Analysis Toolpack(插件函数：分析工具包有24个)
  • Array Formulas: via Sheet.setArrayFormula() and Sheet.removeArrayFormula()(数组公式：通过 Sheet.setArrayFormula() 和 Sheet.removeArrayFormula())
  • Region operators: union, intersection(区域运算符：并集、交集)

Not yet supported(尚不支持)

  • Manipulating table formulas (In Excel, formulas that look like "{=...}" as opposed to "=...")(操作表格的公式（在 Excel 中，公式看起来像“{=...}”而不是“=...”）)
  • Parsing of previously uncalled add-in functions(解析前未调用的加载项函数)
  • Preservation of whitespace in formulas (when POI manipulates them)(公式中保存的空白（当 POI 操作它们时）)

Supported Functions(支持的功能)

To get the list of formula functions that POI supports, you need to call some code!(您可以调用一些代码获取 POI 支持的公式函数列表！)

The methods you need are available on org.apache.poi.ss.formula.eval.FunctionEval. To find which functions your copy of Apache POI supports, use getSupportedFunctionNames() to get a list of the implemented function names. For the list of functions that POI knows the name of, but doesn't currently implement, use getNotSupportedFunctionNames()(您需要的方法可在org.apache.poi.ss.formula.eval.FunctionEval中找到。要想知道您的Apache POI副本支持哪些函数，请使用getSupportedFunctionNames()获取已实现的函数名称列表。对于POI知道名称但尚未实现的函数列表，请使用getNotSupportedFunctionNames())

Internals(内件)

Formulas in Excel are stored as sequences of tokens in Reverse Polish Notation order. The open office XLS spec is the best documentation you will find for the format.(Excel中的公式以逆波兰表示法为顺序存储标记序列。开放式open office XLS spec是您可以找到的格式的最佳文档。)

The tokens used by excel are modeled as individual *Ptg classes in the org.apache.poi.hssf.record.formula package.(excel 使用的标记被建模为 org.apache.poi.hssf.record.formula 包中的单个 *Ptg 类。)

The task of parsing a formula string into an array of RPN ordered tokens is done by the org.apache.poi.ss.formula.FormulaParser class. This class implements a hand written recursive descent parser.(将公式字符串解析为 RPN 有序标记数组的任务由 org.apache.poi.ss.formula.FormulaParser 类完成。这个类实现了一个手工编写的递归下降解析器。)

Formula tokens in Excel are stored in one of three possible operand classes : Reference, Value and Array. Based on the location of a token, its class can change in complicated and undocumented ways. While we have support for most cases, we are not sure if we have covered all bases (since there is no documentation for this area.) We would therefore like you to report any occurrence of #VALUE! in a cell upon opening a POI generated workbook in excel. (Check that typing the formula into Excel directly gives a valid result.)(Excel 中的公式标记存储在三个可能的操作数类之一中：引用、值和数组。根据令牌的位置，它的类可能会以复杂且未记录的方式发生变化。虽然我们支持大多数情况，但我们不确定我们是否涵盖了所有情况（因为没有该领域的文档）。因此，我们希望您报告任何情况：在 Excel 中打开 POI 生成的工作簿时在单元格中出现#VALUE！（检查将公式直接输入Excel是否会给出有效结果）)

Check out the javadocs for details.(查看 javadocs 了解详细信息。)
