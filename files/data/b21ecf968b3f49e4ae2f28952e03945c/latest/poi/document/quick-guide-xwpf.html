POI-XWPF - A Quick Guide(POI-XWPF - 快速指南)

Overview(概述)

XWPF has a fairly stable core API, providing read and write access to the main parts of a Word .docx file, but it isn't complete. For some things, it may be necessary to dive down into the low level XMLBeans objects to manipulate the ooxml structure. If you find yourself having to do this, please consider sending in a patch to enhance that, see the "Contribution to POI" page.(XWPF 有一个相当稳定的核心 API，提供对 Word .docx 文件主要部分的读写访问，但它并不完整。对于某些事情，可能需要深入研究低级别的 XMLBeans 对象来操作 ooxml 结构。如果您发现自己必须这样做，请考虑发送一个补丁来增强它，请参阅“对 POI 的贡献”页面。)

Basic Text Extraction(基本文本提取)

For basic text extraction, make use of org.apache.poi.xwpf.extractor.XWPFWordExtractor. It accepts an input stream or a XWPFDocument. The getText() method can be used to get the text from all the paragraphs, along with tables, headers etc.(对于基本的文本提取，请使用 org.apache.poi.xwpf.extractor.XWPFWordExtractor。它接受输入流或 XWPFDocument。 getText() 方法可用于从所有段落中获取文本，以及表格、标题等。)

Specific Text Extraction(特定文本提取)

To get specific bits of text, first create a org.apache.poi.xwpf.XWPFDocument. Select the IBodyElement of interest (Table, Paragraph etc), and from there get a XWPFRun. Finally fetch the text and properties from that.(要获取特定的文本位，首先创建一个 org.apache.poi.xwpf.XWPFDocument。选择感兴趣的 IBodyElement（表格、段落等），然后从那里获得 XWPFRun。最后从中获取文本和属性。)

Headers and Footers(页眉和页脚)

To get at the headers and footers of a word document, first create a org.apache.poi.xwpf.XWPFDocument. Next, you need to create a org.apache.poi.xwpf.usermodel.XWPFHeaderFooter, passing it your XWPFDocument. Finally, the XWPFHeaderFooter gives you access to the headers and footers, including first / even / odd page ones if defined in your document.(要获取 word 文档的页眉和页脚，首先创建一个 org.apache.poi.xwpf.XWPFDocument。接下来，您需要创建一个 org.apache.poi.xwpf.usermodel.XWPFHeaderFooter，将您的 XWPFDocument 传递给它。最后，XWPFHeaderFooter 让您可以访问页眉和页脚，包括第一页/偶数/奇数页（如果在文档中定义）。)

Changing Text(更改文本)

From a XWPFParagraph, it is possible to fetch the existing XWPFRun elements that make up the text. To add new text, the createRun() method will add a new XWPFRun to the end of the list. insertNewRun(int) can instead be used to add a new XWPFRun at a specific point in the paragraph.(从 XWPFParagraph 中，可以获取构成文本的现有 XWPFRun 元素。要添加新文本，createRun() 方法将在列表末尾添加一个新的 XWPFRun。 insertNewRun(int) 可用于在段落中的特定点添加新的 XWPFRun。)

Once you have a XWPFRun, you can use the setText(String) method to make changes to the text. To add whitespace elements such as tabs and line breaks, it is necessary to use methods like addTab() and addCarriageReturn().(一旦有了 XWPFRun，就可以使用 setText(String) 方法来更改文本。要添加制表符和换行符等空白元素，需要使用 addTab() 和 addCarriageReturn() 等方法。)

Further Examples(更多例子)

For now, there are a limited number of XWPF examples in the Examples Package. Beyond those, the best source of additional examples is in the unit tests. Browse the XWPF unit tests.(目前，示例包中的 XWPF 示例数量有限。除此之外，其他示例的最佳来源是单元测试。浏览 XWPF 单元测试。)
